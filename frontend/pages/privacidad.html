Privacidad

Datos mínimos, uso claro.

Esta versión guarda datos de práctica y recomendaciones en un almacén local del servidor para alimentar el dashboard. No se recopilan pagos ni credenciales.

La web utiliza cookies necesarias para recordar si has confirmado que eres mayor de edad y para guardar tu preferencia de cookies. Estas cookies no se usan para publicidad.

Si aceptas la medición, se activa Vercel Web Analytics para conocer visitas y páginas vistas de forma agregada. Si eliges solo cookies necesarias, el script de analítica no se carga.

En producción, cualquier sistema de cuentas deberá añadir autenticación, política de retención, exportación y borrado de datos de usuario.

Las fuentes y librerías externas cargadas desde CDN pueden recibir metadatos técnicos de navegación según sus propias políticas.
